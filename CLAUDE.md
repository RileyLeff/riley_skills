## Versioning

This is a Claude Code plugin. When committing changes that affect user-facing
behavior (skills, MCP servers, commands, hooks, plugin.json), bump the version
in `.claude-plugin/plugin.json` using semver:

- **patch**: bug fixes, typo corrections
- **minor**: new features, new skills/commands, backwards-compatible changes
- **major**: breaking changes to existing skills or MCP tool signatures
